# SAGE Landing Page

🔒 **Trust Layer for AI Agent Era**

SAGE (Secure Agent Guarantee Engine)의 공식 랜딩 페이지입니다.

## 🚀 기술 스택

- **Framework**: Next.js 15.1.6 (App Router)
- **Language**: TypeScript
- **Styling**: Tailwind CSS 3.4.17
- **Icons**: Lucide React
- **Deployment**: Vercel

## 📦 설치 및 실행

### 패키지 설치

```bash
npm install
```

### 개발 서버 실행

```bash
npm run dev
```

브라우저에서 [http://localhost:3000](http://localhost:3000)을 열어 확인하세요.

### 프로덕션 빌드

```bash
npm run build
npm start
```
